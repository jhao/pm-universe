Power BI 企业原型 角色化导航演示
    角色：数据工程师 连接向导

    新建数据连接

    引导配置数据源类型、凭据与刷新策略。

    完成并创建数据流 返回列表

    步骤 1：选择数据源

    SQL Server Oracle Azure Data Lake Salesforce 自定义 API

    步骤 2：配置凭据

    认证方式

    OAuth2 / 基本身份验证

    凭据有效期

    90 天，提前 7 天提醒

    安全设置

    启用行级安全映射，绑定 Azure AD 组。

    步骤 3：刷新策略

    刷新频率

    每 30 分钟增量刷新

    失败重试

    3 次重试，间隔 5 分钟

    通知

    失败时通知数据工程团队

    页面功能说明

    • 以向导方式串联数据源选择、凭据配置与刷新策略。
    • 强调敏感凭据的有效期与安全策略设置。
    • 完成后可直接进入数据流监控，确认刷新运行状况。
